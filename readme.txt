== Description ==
Enhanced User Stats Tracker allows you to monitor logged-in user activity across your WordPress site.  
Track which pages are being visited, how long users stay, and how they navigate.  
Manage and delete stats by user or date range for better control.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/enhanced-user-stats-tracker/` directory, or install the plugin through the WordPress admin screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure settings under **User Stats > Settings**.

== Frequently Asked Questions ==

= Does it track non-logged-in users? =
No, currently tracking is only for logged-in users.

= Can I delete only selected user stats? =
Yes, you can delete all stats, by specific user, or stats within a certain date range.

= Will it slow down my site? =
No, the plugin is lightweight and optimized for performance.
